Welcome to the User Guide

Hello, Welcome to the DatingApp user guide. Below you will find helpful information on how to guide through the website.

My Profiles

On your “My Profiles” tab you will find all the profiles you have created for your account. You can edit, use, or delete any profile using the buttons found under every profile displayed.

You can also create new profiles for your account by visiting this tab.

Must use profile for access

You must set a current profile by using the use button under a profile you would like to currently use. After you have selected a profile to use as your current profile, you can search for other user profiles, like other user profiles, set up dates with other user profiles, and create blind dates for other profiles.

My Accounts

Click on the My accounts tab to view all your accounts for your current account.

You can use the dashboard on this tab to create new accounts. You can also edit accounts.

After an account is created, the account name cannot be changed. Only the credit card number can be edited and succesfully updated for a unique account.

Favorites

All profiles which you have liked will be displayed on this tab.

You can set up dates from this tab by selecting a date, clicking view full profile, and then clicking set up date.

Dates

You can view all your dates on this tab. All dates for you have been organized by Today, Upcoming, Waiting, Requests, Past, Suggestions, and Blinds.

You can cancel, accept or reject dates by clicking the cancel, accept or reject buttons under every date displayed box.

You can also rate past dates you have been on.

You can also comment on a date by clicking on the comments button under the date display box.

Blind Date

Using this tab, you can create blind dates for other users. You can use the search dashboard on the left side of the screen to find users by physical characteristics, interests, etc.

You can click on select under each profile display box to select this profile as a candidate for your profile date.

Search

You can search for profiles using the search dashboard on the left side of the screen. You can search by physical characteristics, interests, etc. The dating range and dating geo range is automaticly set by the profile you are currently using.

All resulting profiles which meet your desired parameters will be displayed on the right side of the screen.

You can use the View Full Profile to view a more detailed display box of each profile.

You can use the Like button found while viewing a full profile to like a profile which will put the profile into your favorites tab.

You can use the Set up date button to set up a date with the current view full profile.

Welcome to the Manager User Guide

Hello, Welcome to the DatingApp user guide. Below you will find helpful information on how to guide through the website.

Sales Report

You can query for sales reports by going to this tab. You will be presented with a form which will require the needed paramters to generate a sales report.

Users

Below are the following features found in this tab.

All users

Clicking this will generated all users found in the database.

Most Active Users

Clicking this will generated the most active users currently in the database.

Highest Rated User

Clicking this will generate the highest rated user currently in the database.

Find User Who Dated User

You can use the form on the right hand screen to input the first and last name of the user you want to query for. This will generate all users who have dated the user you have entered into the form.

Dates

In this tab, you can use the buttons on the left hand side of the screen to generated the desired reports. All reports will be displayed on the right hand side of the screen.

You can find dates by a given calender date or a given user.

You can also find the date which had the highest rating out of all dates which have occurred.

Revenue

In this tab, you can use the buttons on the left hand side of the screen to generated the desired reports. All reports will be displayed on the right hand side of the screen.

You can generate the total revenues by dates given a calendar date or a user.

You can generate the highest grossing revenue by a user.

You can generate the highest grossing revenue by a customer representative.

Edit Employees

On the left hand screen you will find the edit/create dashboard. On the right hand screen you will find all employees currently in the system.

You can use the buttons Edit and Delete to edit or delete employee information.

You can create new employees as well.

Welcome to the Customer Representative User Guide

Hello, Welcome to the DatingApp user guide. Below you will find helpful information on how to guide through the website.

Suggest

You can use the dashboard found on the left hand side of the screen to find profiles to then suggest for other profiles to date.

You cannot suggest profiles from the same user.

Customers

You can generate customer mailing lists by clicking the Get Mailing List button.

You can generate a suggestion list for a particular customer by clicking the Get Suggestion List.

Select the profile of the customer you want to generate the suggestions list by using the form found on the right hand screen.

You can also book a date by clicking on the Book Date button. Use the form to find dates which have not been booked yet.

Select the date you want to book, and set the booking fee to set the date.

Edit Customers

You can use the dashboard on the left hand screen to edit and create new customers.

The display on the right hand screen displays all current customers in the system.

Use the buttons Edit and Delete to either edit or delete a customer.
